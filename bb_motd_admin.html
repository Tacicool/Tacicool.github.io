Reglas para Admins

  • Respetar a todos los jugadores.
    El trato debe ser siempre cordial y respetuoso con todos los miembros de la comunidad.

  • No abusar de las herramientas administrativas.
    Úsalas con responsabilidad, nunca para perjudicar o alterar injustamente el juego.

  • No construir después del tiempo permitido.
    Se debe respetar el tiempo y reglas de construcción para no alterar la jugabilidad.

Admin responsable = comunidad feliz.
